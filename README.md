# How-To-Build

~~~shell
git clone https://github.com/Reapsn/zmirror-docker.git
cd zmirror-docker/zmirror-runtime/
docker build zmirror-runtime:latest .
cd zmirror-docker/
docker build zmirror:latest .
~~~
